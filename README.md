# Bank Management System - Java & OOP

## Description
A comprehensive banking application developed in Java as part of an Object-Oriented Programming course. The system manages different types of bank accounts and financial operations.

## Key Features
- **Design Patterns:** Implemented the **Observer Pattern** for automated system updates.
- **OOP Principles:** Extensive use of Inheritance, Interfaces, and Polymorphism.
- **Robustness:** Custom Exception Handling for data integrity and user validation.
- **Functionality:** Support for Regular, Business, and VIP accounts with automated fee and profit calculations.

- Java
